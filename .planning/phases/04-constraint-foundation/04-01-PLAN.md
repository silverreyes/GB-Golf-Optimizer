  <files>gbgolf/optimizer/constraints.py</files>
  <behavior>
    All 12 tests in tests/test_constraints.py pass (GREEN state).

    ConstraintSet:
      - dataclass with fields: locked_cards: list[CardKey], locked_golfers: list[str], excluded_cards: list[CardKey], excluded_players: list[str]
      - All fields default to empty list via field(default_factory=list)
      - CardKey = tuple (player: str, salary: int, multiplier: float, collection: str)

    PreSolveError:
      - dataclass with single field: message: str

    check_conflicts(constraints: ConstraintSet) -> PreSolveError | None:
      - Set-intersection: locked_cards keys vs. excluded_cards keys
      - Set-intersection: locked_golfers vs. excluded_players
      - Returns PreSolveError with player name(s) if any conflict found
      - Returns None if no conflicts
      - Check card conflicts first, then golfer conflicts

    check_feasibility(constraints: ConstraintSet, valid_cards: list, config: ContestConfig) -> PreSolveError | None:
      - Build card_map: {(c.player, c.salary, c.multiplier, c.collection): c for c in valid_cards}
      - locked = [card_map[k] for k in constraints.locked_cards if k in card_map] (silently skip missing keys)
      - Check 1 (salary): sum(c.salary for c in locked) > config.salary_max -> PreSolveError with dollar amounts
        Message format: "Locked cards total ${locked_salary:,}. Salary cap is ${config.salary_max:,}. Remove locked cards to proceed."
      - Check 2 (collection): Counter(c.collection for c in locked), for each collection where count > config.collection_limits[coll]:
        Message format: "Locked cards include {count} '{coll}' cards but the limit is {limit}. Remove {count - limit} locked '{coll}' card(s) to proceed."
      - Returns None if all checks pass

    Module exports: __all__ = ["ConstraintSet", "PreSolveError", "check_conflicts", "check_feasibility", "CardKey"]
  </behavior>
