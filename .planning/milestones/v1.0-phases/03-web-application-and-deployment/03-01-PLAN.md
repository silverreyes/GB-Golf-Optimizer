**Step 2: Create gbgolf/web/__init__.py**
Implement `create_app()` factory:
- `Flask(__name__, template_folder="templates", static_folder="static")`
- Wrap with `ProxyFix(app.wsgi_app, x_for=1, x_proto=1, x_host=1, x_prefix=1)`

- Set config_path to `contest_config.json` at project root (resolve relative to this file: `os.path.abspath(os.path.join(os.path.dirname(__file__), "..", "..", "contest_config.json"))`)
- Store as `app.config["CONFIG_PATH"] = config_path`
- Also call `load_config(config_path)` and store as `app.config["CONTESTS"]` (needed by optimize())
- Register blueprint from `gbgolf.web.routes`
- In TESTING mode, skip ProxyFix (it interferes with test client URL generation)

**Step 3: Create gbgolf/web/routes.py**
Blueprint `bp = Blueprint("main", __name__)` with route `@bp.route("/", methods=["GET", "POST"])`.

GET: return `render_template("index.html")`

POST handler:
1. `roster_file = request.files.get("roster")` — if None, return `render_template("index.html", error="Roster file is required")`
2. `projections_file = request.files.get("projections")` — if None, return error
3. Write both to `tempfile.NamedTemporaryFile(delete=False, suffix=".csv", mode="wb")`. Write upload using `file.save(rf)` INSIDE the with block, but capture `.name` for use OUTSIDE.
4. OUTSIDE both with blocks (so files are closed and unlocked on Windows), call:
   - `validation = validate_pipeline(roster_tmp, projections_tmp, config_path)`
   - `result = optimize(validation.valid_cards, current_app.config["CONTESTS"])`
5. `return render_template("index.html", validation=validation, result=result, show_results=True)`
6. Catch `ValueError as e` → `render_template("index.html", error=str(e))`
7. `finally:` unlink both temp files if they exist

**Step 4: Create gbgolf/web/templates/index.html**
Single Jinja2 HTML file implementing ALL the following locked decisions:

- `<details {% if not show_results %}open{% endif %} id="upload-section">` with `<summary>{% if show_results %}Change files{% else %}Upload files{% endif %}</summary>` containing the upload form
- Form: `method="post" enctype="multipart/form-data"` with two file inputs (`name="roster"`, `name="projections"`, both `accept=".csv"`) and a submit button
- Loading overlay: `<div id="loading-overlay" style="display:none;">Optimizing...</div>` with JS: `document.querySelector("form").addEventListener("submit", function() { document.getElementById("loading-overlay").style.display = "flex"; });`
- Error display: `{% if error %}<p class="error">{{ error }}</p>{% endif %}`
- Exclusion report: `{% if validation and validation.excluded %}<section id="exclusion-report">...</section>{% endif %}` with a `<ul>` of `{{ excl.player }} — {{ excl.reason }}`
- Contest sections in fixed order using `{% set contest_order = ["The Tips", "The Intermediate Tee"] %}`:
  - For each contest: `<section><h2>{{ contest_name }}</h2>` then lineup loop
  - Lineup header: `<h3>Lineup {{ loop.index }} — Salary: ${{ lineup.total_salary }} | Proj: {{ "%.2f"|format(lineup.total_projected_score) }}</h3>`
  - Table with thead (Player, Collection, Salary, Multiplier, Proj Score), tbody (one tr per card), tfoot (totals row spanning correctly)
  - Empty lineups list: `{% else %}<p class="infeasible">No lineup could be built for this contest.</p>{% endfor %}`
  - Infeasibility notices: render `result.infeasibility_notices` that contain `contest_name`
